file name:  description

assets: for storing static file for website.

components: contents small react-components.

configs: for any kinds of configuration (like: api configuration).

helpers: for storing helper functions (those functions can repeat in many place)

hooks: for store react hooks

layouts: for store website several layout (like: auth layout, public layout, privet layout).

pages: for store full page of a route.

redux: for store state management.

router: for store page route.

styles: for website styles

validation: for store user input validation
